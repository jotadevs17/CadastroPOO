Cadastro POO - DGT2821
Trabalho prático da disciplina Programação Back-end com Java. O objetivo foi implementar um sistema de cadastro em modo texto com persistência em arquivos binários, utilizando Java puro (sem frameworks) e o padrão DAO/Repository.

Tecnologias
Java (JDK)
NetBeans (Ant)
Persistência em Arquivo (.bin)
Documentação
O relatório completo solicitado pela disciplina está disponível em PDF na pasta de documentação: Acessar Relatório em PDF

Como rodar
Clone o repositório.
Abra o projeto no NetBeans.
Execute a classe CadastroPOO no pacote cadastropoo.
